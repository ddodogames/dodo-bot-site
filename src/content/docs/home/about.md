## What is this
Dodo-Bot is a personal project that started in late 2021. It was mainly made for entertainment with some features added to it. It is mostly made with the goal mindset of reducing the number of bots needed to do each stuff.

## Why
I was fed up with some verified bot in the wild called MEE6 and so i decided to create this project in late 2021. Overtime, it went from sort of competitor attempt to a personal project that i work on it from time to time. The bot was originally going to be multi-purpose but this idea quickly got dropped starting with v2 and eventually i settled on "general-purpose" as the right type for my bot.
